Guia de manipulación de datos de usuarios


Bienvenido sea a las instrucciones de manipulación de usuarios, este es un breve resumen de como se utiliza la página web proporcionada por nuestra organización para crear, leer, actualizar y borrar usuarios en nuestra base de datos.





Acceso a datos

En '(nombre dominio corporativo)/admin' encontrarás una página que pide credenciales de acceso. Si te hemos dado de alta en nuestra organización, deberías saber que credenciales ingresar aquí. Una vez dentro, deberías poder visualizar la tabla de datos 'Users', desde aquí accedes a los datos.


Lectura de datos

Siguiendo los pasos anteriores, deberías visualizar datos de los diferentes usuarios almacenados junto a su información sensible, una barra de búsqueda para filtrar datos, botones de generación de informes (como el de esta guía) y poco más.


Creación de datos

Es probable que te hayas fijado en el botón verde de la esquina superior derecha que pone 'Add User'. Presionando este botón te llevará a una interfaz gráfica amigable en la que ingresas los datos de un nuevo usuario en la base de datos. Una vez hayas acabado, se te regresará a la pantalla anterior y podrás ver a tu usuario agregado correctamente.


Actualización de datos

En la lista de los usuarios de la base de datos, el primer dato sale subrayado indicando que se puede acceder a éste. Presionando ahí te llevará a una pantalla amigable muy parecida a la vista en la creación de datos, en donde se te permitirá editar al usuario seleccionar o deshacer la edición.


Borrado de datos

Es posible que te hayas fijado en el botón rojo de la esquina superior derecha de la pantalla anterior y con las palabras 'Delete User'. Desde la misma pantalla de edición de un usuario se puede eliminar dicho usuario con tan solo un click! Hay que tener en cuenta que esta acción no es reversible.
